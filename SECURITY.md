# Security Policy

## Supported versions

ScaleDown currently supports security updates for the latest published release on `main`.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅ |
| Older   | ❌ |

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, email: **support@scaledown.ai**

Include as much detail as possible:

- affected component(s),
- reproduction steps or proof-of-concept,
- potential impact,
- suggested remediation (if known).

You can expect an acknowledgment within 3 business days. We will coordinate
fixes and disclosure timelines with you.
